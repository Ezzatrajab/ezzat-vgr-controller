# Ändringar i Dashboard-appen - 2026-04-01

## Sammanfattning
Uppdaterat Dashboard-appen för att hämta Rehab-data från rätt källor enligt dina instruktioner:
- Rehab-intäkter från P&L Actual/Budget filer
- Rehab-poäng från KPIer Storg-GBG filen
- Lagt till TeamBesök KPI

## Ändringar i detalj

### 1. Nya funktioner för datainläsning

#### `load_rehab_intakter_from_pl(enhet_kst, manad_str)`
- Läser Rehab-intäkter (3053) från P&L Actual och P&L Budget filer
- Hämtar data för enheter 601 och 602
- Om data saknas eller är 0 i filen, returneras 0

#### `load_kpi_data()`
- Läser KPIer Storg-GBG.xlsx
- Hämtar Rehab-poäng per VC-enhet (102, 103, etc)
- Hämtar TeamBesök per Rehab-enhet (601, 602, etc)
- Om data saknas eller är 0 i filen, returneras 0

#### `uppdatera_rehab_data()`
- Uppdaterar ENHETER_DATA med korrekt data från filerna
- Mappar Rehab-enheter till sina VC-enheter för Rehab-poäng:
  - 601 (Frölunda Torg Rehab) -> 102 (Frölunda Torg VC)
  - 602 (Grimmered Rehab) -> 103 (Grimmered VC)

### 2. Uppdaterad Rehab-visning

**Tidigare:**
- Rehab-intäkter: Hårdkodade värden
- Rehab-poäng: Beräknat från intäkter (intäkt / 523)
- Ingen TeamBesök KPI

**Nu:**
- **Rehab-intäkter**: Hämtas från P&L Actual vs P&L Budget
- **Rehab-poäng**: Hämtas från KPIer Storg-GBG (från motsvarande VC-enhet)
- **TeamBesök**: Ny KPI hämtad från KPIer Storg-GBG

### 3. Layout-ändringar för Rehab-enheter

**Ny struktur:**

**Rad 1 - Data från filer:**
- Kolumn 1: 💰 Rehab Intäkter (P&L) - från P&L Actual/Budget
- Kolumn 2: 💪 Rehab Poäng (KPI) - från KPIer Storg-GBG
- Kolumn 3: 🏥 TeamBesök (KPI) - från KPIer Storg-GBG

**Rad 2 - Personal:**
- Kolumn 4: 👔 FTE Total
- Kolumn 5: 💰 Personalkostnad

### 4. VC-enheter uppdaterade

För VC-enheter (102, 103) visas nu också Rehab-poäng från KPI-filen istället för att beräknas.
